# AuroraWeat# 🌌 AuroraWeather | 极光天气

一款基于 Python 开发的轻量化实时天气预报工具。该项目不仅是一个实用工具，更是我探索 **AI 辅助编程 (AI-Augmented Development)** 与 **自动化工作流** 的实践成果。

## ✨ 项目亮点
* **实时数据抓取**：集成天气 API，实现全球城市天气数据的秒级获取。
* **AI 协作开发**：本项目全流程深度利用 AI 工具进行代码架构设计、逻辑优化及 Debug，开发效率提升约 300%。
* **简洁交互**：注重用户体验，提供直观的命令行/界面展示。

## 🛠️ 技术栈
* **Language**: Python 3.x
* **APIs**: 和风天气 / 高德天气 API
* **Workflow**: AI Prompt Engineering + Git Version Control

## 🚀 快速开始
1. 克隆项目：`git clone https://github.com/Lin-JJ9/AuroraWeather.git`
2. 安装依赖：`pip install -r requirements.txt` (如果你有依赖文件)
3. 运行程序：`python main.py`
